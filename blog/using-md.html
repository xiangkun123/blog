# markdown 写法

# 1、标题

示例代码：

# 标题一
## 标题二
### 标题三
#### 标题四
##### 标题五
1
2
3
4
5

效果：

# 标题一

# 标题二

# 标题三

# 标题四

# 标题五

# 2、列表

列表分为有序列表和无序列表

无序列表示例代码：

- 苹果
- 香蕉
- 梨子
1
2
3

效果：

  • 苹果
  • 香蕉
  • 梨子

有序列表示例代码：

1. 苹果
2. 香蕉
3. 梨子
1
2
3

效果：

  1. 苹果
  2. 香蕉
  3. 梨子

# 9、语法高亮

vuepress 使用了 Prism 来为 markdown 实现语法高亮。你只需在代码块的开始倒勾中附加一个有效的语言别名。

没有设置之前示例代码：

function hello() {
  console.log("hello vuepress");
}
hello();
1
2
3
4

设置

``` javascript
function hello() {
  console.log("hello vuepress");
}
hello();
```
1
2
3
4
5
6

效果：

function hello() {
  console.log("hello vuepress");
}
hello();
1
2
3
4

如果要给特定的行高亮，在语言别名后加上一个行数

``` javascript{2}
function hello() {
  console.log("hello vuepress");
}
hello();
```

# 10、使用组件

所有在 .vuepress/components 中找到的 *.vue 文件将会自动地被注册为全局的异步组件，如：

.
└─ .vuepress
   └─ components
      ├─ demo-component.vue
      ├─ OtherComponent.vue
      └─ Foo
         └─ Bar.vue
1
2
3
4
5
6
7

你可以在任意的 Markdown 文件中使用这些组件（组件名是通过文件名取到的）：

<demo-component />
<OtherComponent />
<Foo-bar />
1
2
3

效果：

Hello this is <demo-component>

This is another component

Hello this is <Foo-Bar>
